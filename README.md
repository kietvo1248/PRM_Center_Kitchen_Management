# PRM Center Kitchen Management - VFC Fried Chicken

## 📌 Giới thiệu dự án
Dự án **Center Kitchen Management** là một ứng dụng Android được thiết kế để quản lý hoạt động của hệ thống nhà bếp trung tâm cho chuỗi cửa hàng gà rán **VFC**. Ứng dụng hỗ trợ các vai trò khác nhau trong quy trình vận hành, từ quản lý đơn hàng, kho bãi đến điều phối cung ứng.

## 🚀 Tính năng chính
- **Đăng nhập & Phân quyền:** Hỗ trợ nhiều vai trò như Admin, Manager, và Supply Coordinator với các giao diện và quyền hạn riêng biệt.
- **Quản lý thực đơn:** Cho phép quản lý các món ăn trong thực đơn của nhà bếp.


## 🛠 Công nghệ sử dụng
- **Ngôn ngữ:** Java / Kotlin
- **Kiến trúc:** MVVM (Model-View-ViewModel)
- **Thư viện mạng:** Retrofit 2, OkHttp 3, Gson
- **Giao diện:** XML Layout, Material Design components, View Binding
- **Quản lý ảnh:** Glide
- **Lưu trữ:** SharedPreferences (SessionManager)

## 📋 Yêu cầu hệ thống
- Android Studio Ladybug hoặc mới hơn.
- JDK 17+.
- Android SDK 24 (Android 7.0) trở lên.

## ⚙️ Cài đặt
1. **Clone project:**
   ```bash
   git clone <repository-url>
   ```

 
2. **Build project:**
   Chọn `File > Sync Project with Gradle Files` trong Android Studio.
